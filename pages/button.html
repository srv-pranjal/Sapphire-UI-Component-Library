Buttons

A clickable button is used to interact with the user. Buttons can be used in different ways like navigating to different pages/sections, submit forms, open modals, etc.

Solid Buttons

There are different types of solid buttons available which comprises of Primary and Secondary buttons, success button, and error button. These buttons can be used directly by adding the class btn along with the additional button type class - btn--primary, btn--secondary, btn--success, btn--error. You can also disable any button by just including the class - btn--disabled along with any button type.
You can copy example html code from below code snippet. 


    <button class="btn btn--primary">Primary Button</button>

    <button class="btn btn--secondary">Secondary Button</button>

    <button class="btn btn--success">Success</button>

    <button class="btn btn--error">Error</button>
    
    <button class="btn btn--primary btn--disabled">Disabled</button>

Outlined Buttons

Sometimes we need buttons with transparent background-color. For this, there are 4 types of outlined buttons available. These buttons can be used directly by replacing the default modifier classes with btn--outline-primary, btn--outline-secondary, btn--outline-success, btn--outline-error to remove background colors on buttons. 


    <button class="btn btn--outline-primary">Primary Button</button>

    <button class="btn btn--outline-secondary">Secondary Button</button>

    <button class="btn btn--outline-success">Success</button>
    
    <button class="btn btn--outline-error">Error</button>

Link Buttons

Links can be used as buttons by using the standard button classes as mentioned above, enclosed in anchor tag along with role="button" to specify the purpose of links. A text link button can be built by using the classes btn btn--link-text.
Check out below examples of Link Buttons

Solid Link Button Outlined Link Button Text Link Button 

    <a class="btn btn--primary" role="button" href="#">Solid Link Button</a>
    
    <a class="btn btn--outline-primary" role="button" href="#">Outlined Link Button</a>

    <a href="#" class="btn btn--link-text" role="button">Text Link Button</a>

Icon Buttons

Buttons can also have icons. This can be built by using the classes mentioned above along with the icons embedded in i tag with class name - btn__icon. Check out the below examples which are using icons. 


    <button class="btn btn--primary">
        <i class="fa fa-home btn__icon"></i>
        Home
    </button>

    <button class="btn btn--outline-primary">
        <i class="fa fa-trash btn__icon"></i>
        Trash
    </button>

    <button class="btn btn--success">
        <i class="fa fa-check-circle btn__icon"></i>
        Accept
    </button>

    <button class="btn btn--outline-error">
        <i class="fa fa-close btn__icon"></i>
        Reject
    </button>

Floating Action Buttons

A floating action button (FAB) performs the primary, or most common, action on a screen. A FAB can be built using the standard button classes mentioned above along with the class - btn--float. 


    <button class="btn btn--primary btn--float">
        <i class="fa fa-arrow-up btn__icon"></i>
    </button>

    <button class="btn btn--outline-secondary btn--float">
        <i class="fa fa-camera btn__icon"></i>
    </button>

    <button class="btn btn--outline-primary btn--float">
        <i class="fa fa-share btn__icon"></i>
    </button>

    <button class="btn btn--secondary btn--float">
        <i class="fa fa-envelope btn__icon"></i>
    </button>
