Avatar

Avatars are components, that are typically used to display a user profile as a picture, an icon, or short text.

Image Avatar

Avatars can be used directly by including the class - avatar along with the class name for the size of avatar needed. There are 4 sizes available for avatar - avatar--xs, avatar--sm, avatar--md, avatar--lg.
Avatars can also have image with class name - avatar__img as shown below in the code snippet 


      <div class="avatar avatar--xs">
        <img class="avatar__img"
          src="https://i.pravatar.cc/300?img=8"
          alt="Avatar X-Small" />
      </div>

      <div class="avatar avatar--sm">
        <img class="avatar__img"
          src="https://i.pravatar.cc/300?img=4"
          alt="Avatar Small" />
      </div>

      <div class="avatar avatar--md">
        <img class="avatar__img"
          src="https://i.pravatar.cc/300?img=1"
          alt="Avatar Medium" />
      </div>

      <div class="avatar avatar--lg">
        <img class="avatar__img"
          src="https://i.pravatar.cc/300?img=2"
          alt="Avatar Large" />
      </div>

Icon Avatar

Using icons as avatar content is very easy. Just embed the icons in i tag and use it in an avatar with the classes mentioned above. Along with the size, you can also select the color of your avatar by using the classes - avatar--primary, avatar--secondary, avatar--success and avatar--danger.
Check out the below examples for using icons as avatar content. 


    <div class="avatar avatar--xs avatar--primary">
      <i class="fa fa-user-o"></i>
    </div>

    <div class="avatar avatar--sm avatar--secondary">
      <i class="fa fa-user"></i>
    </div>

    <div class="avatar avatar--md avatar--success">
      <i class="fa fa-user-circle"></i>
    </div>

    <div class="avatar avatar--lg avatar--danger">
      <i class="fa fa-users"></i>
    </div>

Text Avatar

For users who don't use their profile pictures, the initials of their name can be used as avatar content. A text avatar can be created by simply adding the text in the avatar. Here also you have a choice to choose the size and color of avatar by using the classes mentioned above. For eg. - avatar avatar-sm avatar--secondary classes will be used to create a small sized avatar of secondary color.
