Alert

An alert displays a short, important message in a way that attracts the user's attention without interrupting the user's task.

Standard Alerts

There are 6 types of available alerts which comprise of standard light and dark alerts, success alerts, error alerts, warning alerts and info alerts. These alerts can be used directly by adding the class alert along with the additional alert type class - alert--dark, alert--light, alert--success, alert--danger, alert--warning, alert--info.
You can copy example html code from below code snippet.

A standard dark alert
A standard light alert
A standard success alert
A standard danger alert
A standard warning alert
A standard info alert 

    <div class="alert alert--dark" role="alert">
      A standard dark alert
    </div>

    <div class="alert alert--light" role="alert">
      A standard light alert
    </div>
          
    <div class="alert alert--success" role="alert">
      A standard success alert
    </div>

    <div class="alert alert--danger" role="alert">
      A standard danger alert
    </div>

    <div class="alert alert--warning" role="alert">
      A standard warning alert
    </div>

    <div class="alert alert--info" role="alert">
      A standard info alert
    </div>

Alerts With Icon

Alerts can also have icons. This can be built by using the classes mentioned above along with the icons embedded in i tag.
Check out the below examples which are using icons in the success, error, warning and info alerts.

A success alert with icon
A danger alert with icon
A warning alert with icon
A info alert with icon 

    <div class="alert alert--success" role="alert">
      <i class="fa fa-check-circle"></i>
      A success alert with icon
    </div>

    <div class="alert alert--danger" role="alert">
      <i class="fa fa-exclamation-circle"></i>
      A danger alert with icon
    </div>

    <div class="alert alert--warning" role="alert">
      <i class="fa fa-warning"></i>
      A warning alert with icon
    </div>
    
    <div class="alert alert--info" role="alert">
      <i class="fa fa-info-circle"></i>
      A info alert with icon
    </div>

Alerts With Close Button

Alerts can also have a close button with the class name alert__btn-close as shown in the code snippet below.

A sucess alert with close button 

    <div class="alert alert--warning" role="alert">
      <i class="fa fa-warning"></i>
      A warning alert with icon
      <i class="fa fa-close alert__btn-close"></i>
    </div>
