Toast

Toast can be used to provide brief messages about app processes at the bottom of the screen.

Basic Toast

There are 2 types of toasts available - left aligned and right aligned toasts. These toasts can be used directly by adding the class toast along with the additional toast type class - toast--left, toast--right. Check out the Live Demo by clicking on the Demo button.

Your photo has been archieved.

UNDO

Can't send photo. Retry in 5 seconds.

RETRY 

      <div id="toast--left" class="toast toast--left">
        <p class="toast__txt">Your photo has been archieved.</p>
        <a class="btn btn--link-text" role="button">UNDO</a>
      </div>

      <div id="toast--right" class="toast toast--right">
        <p class="toast__txt">Can't send photo. Retry in 5 seconds.</p>
        <a class="btn btn--link-text">RETRY</a>
      </div>

Toast With Close Button

Toasts can also have a close button with the class name toast__btn-close as shown in the code snippet below. Check out the Live Demo by clicking on the Demo button.

Your photo has been archieved.

UNDO × 

      <div id="toast--demo" class="toast toast--left">
        <p class="toast__txt">Your photo has been archieved.</p>
        <a class="btn btn--link-text" role="button">UNDO</a>
        <a class="toast__btn-close">×</a>
      </div>
