Typography

Typography is the way of arranging and presenting texts.

Headings

There are 6 types of heading available from heading--1 to heading--6 where heading--1 is the largest and heading--6 being the smallest. These can be used by directly using the class name - heading--1, heading--2, heading--3, heading--4, heading--5, heading--6.

Heading-1

Heading-2

Heading-3

Heading-4

Heading-5
Heading-6

    <h1 class="heading-1">Heading-1</h1>

    <h2 class="heading-2">Heading-2</h2>

    <h3 class="heading-3">Heading-3</h3>

    <h4 class="heading-4">Heading-4</h4>

    <h5 class="heading-5">Heading-5</h5>

    <h6 class="heading-6">Heading-6</h6>

Text utilities

Text Utilities are available in 3 sizes - Large, Medium and Small. These utilities can be used directly by using the class names - txt--lg, txt--md, txt--sm. Check out the below implementaion for more details.

Large Text

Medium Text

Small Text

    <p class="txt--md">Medium Text</p>

    <p class="txt--sm">Small Text</p>

Colored Text Utilities

Text Utilities are also available in 3 different colors - Primary, Secondary and Gray. These utilities can be used directly by using the class names - txt--primary, txt--secondary, txt--gray along with the desired class for size of text. Check out the below implementaion for more details.

Primary Colored Text

Secondary Colored Text

Gray Colored Text


    <p class="txt--primary txt--lg">Primary Colored Text</p>

    <p class="txt--secondary txt--lg">Secondary Colored Text</p>

    <p class="txt--gray txt--lg">Gray Colored Text</p>

Aligned Text Utilities

Text Utilities can be either left aligned, center aligned or right aligned. These utilities can be used directly by using the class names - txt--left, txt--center, txt--right along with the desired class for size of text. Check out the below implementaion for more details.

Center Aligned Text
