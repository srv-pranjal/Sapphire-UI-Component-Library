Image

Images can be responsive to fit the parent's width, and also can be customised to be round shaped

Responsive Image

An image can be made responsive by including the class name - img-responsive to the image.
Check out the below example for responsive image. 


    <img
      class="img-responsive"
      src="../assets/component-library-illustration.svg"
      alt="Responsive Image"
    />

Rounded Image

A rounded image can be built by using the class name - img-round in the img tag.
Check out the below example for rounded image 


    <img
      class="img-round"
      src="https://i.pravatar.cc/300?img=2"
      alt="Responsive Image"
    />
