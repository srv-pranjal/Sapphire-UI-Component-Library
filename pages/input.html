Input

Inputs are used to create interactive controls for web-based forms in order to accept data from the user.

Text Input

Two types of text inputs are available which includes - Standard and Outlined text fields. These input fields can be built directly by including the class name - input along with the class name for the type of input fields - input--standard input--outlined
Check out below examples for text fields. 


    <input type="text" class="input input--standard"
      placeholder="Standard"
    />

    <input type="text" class="input input--outlined"
      placeholder="Outlined"
    />

Validation Input

For both Standard and Outlined input, 3 validation inputs are available which includes - Success, Warning and Error. Success Inputs can be used in case the password matches with the confirm password, Warning Inputs can be used in case the entered password is not strong enough and Error Inputs can be used if the entered username/password is incorrect or entered password doesn't match with the confirm password.
To start using validation inputs for standard input, include the class names - input--standard-success, input--standard-warning, input--standard-error and for outlined input, include the class names - input--outlined-success, input--outlined-warning, input--outlined-error along with the class names mentioned above. 


    <input class="input input--outlined input--outlined-warning"
      type="password"
      placeholder="Weak password"
    />

    <input class="input input--outlined input--outlined-error"
      type="password"
      placeholder="Invalid Password"
    />
    
    <input class="input input--outlined input--outlined-success"
      type="password"
      placeholder="Password matches"
    />


    <input class="input input--standard input--standard-warning"
      type="password"
      placeholder="Weak password"
    />

    <input class="input input--standard input--standard-error"
      type="password"
      placeholder="Invalid Password"
    />

    <input class="input input--standard input--standard-success"
      type="password"
      placeholder="Password matches"
    />
