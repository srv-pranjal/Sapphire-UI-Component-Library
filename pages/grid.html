Grid

Simplified Grids can be used to structure elements in rows and columns.

Grid with 2 columns

A Simple Grid layout with 2 columns can be built by including the class - grid-col-2 along with the default class name - grid. For giving gaps among grid items gap-1 class can be used. Checkout the below code for implementing a Two column grid.

Grid Item 1
Grid Item 2

      <div class="grid grid-col-2 gap-1">
        <div class="grid__item">Grid Item 1</div>
        <div class="grid__item">Grid Item 2</div>
      </div>

Grid with 2 rows

A Grid layout with 2 rows can be built by including the class - grid-row-2 along with the default class name - grid. Checkout the below code for implementing a Two row grid.

Grid Item 1
Grid Item 2

      <div class="grid grid-row-2 gap-1">
        <div class="grid__item">Grid Item 1</div>
        <div class="grid__item">Grid Item 2</div>
      </div>
