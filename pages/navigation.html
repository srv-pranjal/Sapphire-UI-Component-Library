Navigation

Navigation bar is used to navigate through the web apps or websites.

Simple Navigation

A simple navigation bar can be build by using a nav with class name nav wrapped inside a container with class name - wrapper.
Check out the below code to get started with a simple navigation bar.

MyWebsite

  • Home
  • Profile
  • Cart
  • Wishlist 

    <header class="wrapper">
      <h3 class="wrapper__title">MyWebsite</h3>
      <nav class="nav">
        <ul class="nav__list">
          <li class="nav__list-item">
            <a href="#" role="button">Home</a>
          </li>
          <li class="nav__list-item">
            <a href="#" role="button">Profile</a>
          </li>
          <li class="nav__list-item">
            <a href="#" role="button">Cart</a>
          </li>
          <li class="nav__list-item">
            <a href="#" role="button">Wishlist</a>
          </li>
        </ul>
      </nav>
    </header>

Navigation with Search Bar

A navigation bar with search bar can be implemented by including a container with class name - nav__search-bar as shown below in the code.

MyWebsite

  • Home
  • Profile
  • Cart
  • Wishlist 

    <header class="wrapper">
      <h3 class="wrapper__title">MyWebsite</h3>
      <nav class="nav">
        <div class="nav__search-bar">
          <input
            type="text"
            class="input input--outlined"
            placeholder="Search"
          />
          <i class="fa fa-search"></i>
        </div>
        <ul class="nav__list">
          <li class="nav__list-item">
            <a href="#" role="button">Home</a>
          </li>
          <li class="nav__list-item">
            <a href="#" role="button">Profile</a>
          </li>
          <li class="nav__list-item">
            <a href="#" role="button">Cart</a>
          </li>
          <li class="nav__list-item">
            <a href="#" role="button">Wishlist</a>
          </li>
        </ul>
      </nav>
    </header>
