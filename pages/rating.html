Rating

Rating can be used to show ratings on the product, or it can also be used for taking ratings from the user on an e-commerce website.

Static Rating

A Static rating, to display rating of a product, can be built by using the class name - rating. Selected stars are to be given the class - rating__star--selected and rest are to be given - rating__star--disabled along with the default class name - rating__star. Check out the below implementation of a Static Rating. 


    <div class="rating">
        <label>
            <i class="fa fa-star rating__star rating__star--selected"></i>
        </label>
        <label>
            <i class="fa fa-star rating__star rating__star--selected"></i>
        </label>
        <label>
            <i class="fa fa-star rating__star rating__star--selected"></i>
        </label>
        <label>
            <i class="fa fa-star rating__star rating__star--disabled"></i>
        </label>
        <label>
            <i class="fa fa-star rating__star rating__star--disabled"></i>
        </label>
    </div>

Live Rating

A Live rating, to take feedback, can be built by using the class name - rating--live along with the default class name - rating. Check out the below implementation of a Live Rating. 


    <div id="example-rating" class="rating rating--live">
        <label>
            <i class="fa fa-star rating__star rating__star--selected"></i>
        </label>
        <label>
            <i class="fa fa-star rating__star rating__star--selected"></i>
        </label>
        <label>
            <i class="fa fa-star rating__star rating__star--disabled"></i>
        </label>
        <label>
            <i class="fa fa-star rating__star rating__star--disabled"></i>
        </label>
        <label>
            <i class="fa fa-star rating__star rating__star--disabled"></i>
        </label>
    </div>
