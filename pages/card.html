Cards

Cards are containers which can be used to display few short, related pieces of information.

Vertical Card

Cards can be used to display products on an e-commerce website. A vertical card can be built by including the class name - card--vertical along with the default class card.
Copy below code and put your desired text and image and you are good to go.

White T-shirt

by Sapphire Designs
White solid T-shirt, has a round neck and short sleeves.
₹399 ₹299 25% OFF 

    <article class="card card--vertical">
      <div class="card__content">
        <div class="card__img-container">
          <img src="../assets/t-shirt.jpg" class="img-responsive" />
        </div>
        <div class="card__product-details">
          <div class="card__title">
            <h2>White T-shirt</h2>
            <small>by Sapphire Designs </small>
          </div>
          <div class="card__description">
            White solid T-shirt, has a round neck and short sleeves.
            <div class="card__price-wrapper">
              <span class="card__price-original">₹399</span>
              <span class="card__price-discounted">₹299</span>
              <span class="card__discount highlight">25% OFF</span>
            </div>
          </div>
        </div>
      </div>
      <div class="card__footer">
        <button class="btn btn--outline-secondary">
          <i class="fa fa-heart-o"></i>
        </button>
        <button class="btn btn--outline-secondary">
          <i class="fa fa-shopping-cart"></i>
        </button>
        <button class="btn btn--primary">Buy Now</button>
      </div>
    </article>
