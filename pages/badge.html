Badges

Badges are components that can be used to display either status of the user or to show notification count.

Badge on Avatar

There are two types of badges available to display user's status (online/offline). To use a badge, wrap your avatar(Image/Icon/Text) in a container with class name - badge along with a span tag with class name - badge-status. Each size of avatar have their respective sizes of badges. You can select the size of avatar by including class name - badge__status--lg, badge__status--md, badge__status--sm, badge__status--xs. For selecting the type of status,you can include the class name - badge__status--online, badge__status--offline for online and offline respectively.
Check out the below examples to understand how to build a badge on avatars.

    </div>

Badges on icon

For using badges on icon as notification count, just wrap the icon in a container with class name - badge along with a span tag with class name - badge__status badge__number.
Check out the below example of badges on icons.

1
